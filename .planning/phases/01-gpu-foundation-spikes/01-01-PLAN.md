       - pub async fn init_gpu() -> Result<(wgpu::Device, wgpu::Queue), GpuError>

       - Use wgpu::Limits::default(), wgpu::Features::empty()
       - Label device "velos-gpu"
       - Add pub fn try_init_gpu_sync() -> Option<(Device, Queue)> convenience wrapper using pollster for tests

    2. Implement crates/velos-gpu/src/buffer_pool.rs:
       - pub struct BufferPair { front: wgpu::Buffer, back: wgpu::Buffer }
       - BufferPair::new(device, size, label) with STORAGE | COPY_SRC | COPY_DST usage flags
       - BufferPair::swap(&mut self) swaps front/back via std::mem::swap
       - BufferPair::front(&self) -> &Buffer and BufferPair::back(&self) -> &Buffer accessors
       - pub struct BufferPool managing named BufferPairs, with get(name) -> &BufferPair

    3. Implement crates/velos-gpu/src/shader_registry.rs:
       - pub struct ShaderRegistry caching wgpu::ComputePipeline by name
       - load_shader(device, name, wgsl_source) -> Result<(), GpuError>
       - get_pipeline(name) -> Option<&ComputePipeline>
       - Bind group layout creation for uniform params + storage buffers

    4. Implement crates/velos-gpu/src/dispatcher.rs:
       - pub struct ComputeDispatcher { device, queue }
       - dispatch(pipeline, bind_group, workgroup_count) encodes and submits
       - readback(source_buffer, size) -> Vec<u8> copies to staging buffer, maps, reads
       - Workgroup count helper: ceil_div(agent_count, 256)

    5. Implement crates/velos-gpu/src/index_map.rs:
       - pub struct IndexMap { gpu_to_entity: Vec<hecs::Entity>, entity_to_gpu: HashMap<hecs::Entity, u32> }
       - rebuild(world: &hecs::World) iterates Position components, assigns dense GPU indices
       - len(), entities() -> iterator, gpu_index(entity) -> Option<u32>, entity(gpu_idx) -> Option<Entity>

    6. Create crates/velos-gpu/shaders/fixed_point.wgsl:
       - struct I64 { hi: i32, lo: u32 } for emulated 64-bit
       - fn mul_i32_to_i64(a: i32, b: i32) -> I64: split each operand into 16-bit halves (unsigned), multiply four partial products, reassemble with carry propagation, handle sign correction for negative inputs (negate if signs differ)
       - fn shr_i64(v: I64, shift: u32) -> i32: right-shift emulated i64 by shift bits, return low i32
       - fn fix_mul_q16_16(a: i32, b: i32) -> i32: mul_i32_to_i64(a,b) then shr_i64 by 16
       - fn fix_mul_q12_20(a: i32, b: i32) -> i32: mul_i32_to_i64(a,b) then shr_i64 by 20
       - fn fix_div_q16_16(a: i32, b: i32) -> i32: emulated division via shift-before-divide pattern using emulated i64 (left-shift a by 16 bits in i64, then divide by b)
       - Validation compute kernel: @compute @workgroup_size(256) fn validate_fixed_point: reads input pairs from storage buffer, writes results to output buffer using fix_mul and fix_div

    7. Create crates/velos-gpu/shaders/round_trip.wgsl:
