1. Implement `crates/velos-api/src/camera.rs`:
   - `Camera` struct: id (u32), lat (f64), lon (f64), heading_deg (f32), fov_deg (f32), range_m (f32), name (String), covered_edges (Vec<u32>)
   - `CameraRegistry` struct: cameras (HashMap<u32, Camera>), next_id (AtomicU32 or u32 with &mut self)
   - `CameraRegistry::register(&mut self, request: &RegisterCameraRequest, edge_tree: &RTree<EdgeSegment>, projection: &EquirectangularProjection) -> Camera`:
     - Assign next ID, convert lat/lon to local meters via projection
     - Call `edges_in_fov()` to find covered edges
     - Store and return Camera
   - `CameraRegistry::get(&self, id: u32) -> Option<&Camera>`
   - `CameraRegistry::list(&self) -> Vec<&Camera>`
   - `CameraRegistry::contains(&self, id: u32) -> bool`
   - `edges_in_fov(cam_pos: [f64; 2], heading_rad: f64, half_angle_rad: f64, range_m: f64, tree: &RTree<EdgeSegment>) -> Vec<u32>`:
     - Bounding box query: circle of radius range_m around cam_pos
     - For each segment in bounding box, compute midpoint, check distance < range_m
     - Compute angle to segment midpoint, normalize difference to [-PI, PI], check < half_angle
     - Deduplicate edge IDs
   - Unit tests per behavior list above (use manually-constructed RTree with known edge positions for deterministic testing)

2. Implement `crates/velos-api/src/aggregator.rs`:
   - `VehicleClass` enum mirroring proto (Motorbike, Car, Bus, Truck, Bicycle, Pedestrian) with From<i32> conversion from proto enum
   - `TimeWindow` struct: start_ms (i64), end_ms (i64), counts (HashMap<VehicleClass, u32>), speed_samples (HashMap<VehicleClass, (f32, u32)>) where tuple is (sum_speed*count, total_count)
   - `TimeWindow::mean_speed(&self, class: VehicleClass) -> Option<f32>` -- sum / count if count > 0
   - `DetectionAggregator` struct: window_duration_ms (i64, default 300_000), retention_ms (i64, default 3_600_000), cameras (HashMap<u32, Vec<TimeWindow>>)
   - `DetectionAggregator::new(window_duration_ms: i64, retention_ms: i64) -> Self`
   - `DetectionAggregator::default() -> Self` -- 5-min windows, 1-hr retention
   - `DetectionAggregator::ingest(&mut self, camera_id: u32, event: &DetectionEvent)`:
     - Find or create window for event.timestamp_ms
     - Accumulate count, accumulate speed sample if present
   - `DetectionAggregator::gc(&mut self, now_ms: i64)` -- remove windows with end_ms <= now_ms - retention_ms
   - `DetectionAggregator::latest_window(&self, camera_id: u32) -> Option<&TimeWindow>` -- most recent window for a camera
   - `DetectionAggregator::total_count(&self, camera_id: u32, class: VehicleClass) -> u32` -- sum across all retained windows
   - Unit tests per behavior list above
