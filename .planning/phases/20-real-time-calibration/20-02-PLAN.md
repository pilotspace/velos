<task type="auto">
  <name>Task 1: Enhance calibration panel with status, details, and pause toggle</name>
  <files>crates/velos-gpu/src/app_egui.rs</files>
  <action>
    Rewrite `draw_calibration_panel` to include three sections:

    **1. Header with status indicator and pause toggle:**
    - Status indicator using colored circle (egui painter rect_filled with rounding):
      - "Calibrating" (green, rgb 0,200,0) when: cameras registered AND at least one camera has consecutive_stale_windows < 3 AND calibration_paused is false
      - "Idle" (gray, rgb 128,128,128) when: no cameras registered
      - "Stale" (yellow, rgb 230,200,0) when: cameras registered AND ALL cameras have consecutive_stale_windows >= 3
      - "Paused" (orange, rgb 230,130,0) when: calibration_paused is true (override other states)
    - Pause/resume checkbox: `ui.checkbox(&mut sim.calibration_paused, "Pause Calibration")` -- directly toggles the field that step_calibration checks
    - gRPC address label (existing)

    **2. Global summary row:**
    - "Active cameras: {N}" (count of cameras with consecutive_stale_windows < 3)
    - "Mean ratio: {:.2}" (average previous_ratio across all cameras, or 1.00 if none)
    - "Last calibration: {secs}s ago" (sim.sim_time - sim.last_calibration_time, formatted as integer seconds)

    **3. Per-camera grid (enhanced from existing):**
    - Columns: Camera | Obs | Sim | Ratio | Status
    - Camera: cam.name (existing)
    - Obs: state.last_observed (existing)
    - Sim: state.last_simulated (existing)
    - Ratio: state.previous_ratio formatted to 2 decimal places (existing)
    - Status: colored text indicator
      - "Live" (green) when consecutive_stale_windows == 0
      - "Stale ({N})" (yellow) when consecutive_stale_windows 1-2
      - "Decaying" (orange) when consecutive_stale_windows >= 3

    Layout: use same pattern as existing draw_calibration_panel (lock registry, iterate cameras, read calibration_states). Clone needed data out of locks before rendering (keep lock scope minimal per existing pattern).

    Egui styling at Claude's discretion -- keep it simple, functional, consistent with existing panel sections.

    IMPORTANT: The `sim` parameter must be `&mut SimWorld` (not `&SimWorld`) to allow the pause checkbox to mutate `calibration_paused`. Update the function signature and the call site in draw_control_panel.
  </action>

    - Per-camera rows show staleness status column
