       b. `impl SimWorld { pub(crate) fn step_meso(&mut self, dt: f64) }`:
          - Early return if `!self.meso_enabled`
          - Call try_exit() on each SpatialQueue; collect exiting vehicles
          - For each exiting vehicle: call `spawn_from_meso()` to reconstruct ECS entity
          - No entry logic here -- entry happens in advance_to_next_edge() interception

       c. `impl SimWorld { fn spawn_from_meso(&mut self, vehicle: MesoVehicle) }`:
          - Look up MesoAgentState from `self.meso_agent_states` using vehicle.vehicle_id
          - Compute insertion speed via `velocity_matching_speed(meso_exit_speed, last_micro_speed)`
          - `meso_exit_speed`: derive from SpatialQueue travel_time and edge length (speed = length / travel_time)
          - `last_micro_speed`: query agents on the exit edge, find the last (lowest offset) agent's speed, default to edge free-flow speed if no agents
          - Spawn ECS entity at buffer zone start (offset_m = 0.0, lane = 0) with preserved Route, VehicleType, etc.
          - Remove vehicle_id from meso_agent_states after spawn
          - Log warning if meso_agent_states doesn't contain the vehicle_id (identity loss pitfall)

       d. `impl SimWorld { fn check_gap_for_insertion(&self, edge_id: u32) -> bool }`:
          - Check if there's physical space at the start of the micro edge for insertion
          - If nearest agent on the edge has offset_m < 10.0 (vehicle length + buffer), return false
          - If false, vehicle stays in meso queue for one more frame (held at exit)

    2. **Intercept micro-to-meso transitions** (in `sim_helpers.rs` `advance_to_next_edge()`):
       - After computing next_info (next edge and step), before applying the edge transition:

         - Store MesoAgentState in `self.meso_agent_states`
         - Mark route as complete to trigger removal in remove_finished_agents() (cleanest despawn path)
         - Log at debug level: "Agent {} entering meso zone on edge {}"
