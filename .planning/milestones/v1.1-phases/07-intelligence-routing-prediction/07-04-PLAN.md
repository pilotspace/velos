    pub struct HistoricalMatcher {
        // Flat Vec indexed by [edge_idx * 24 * 4 + hour * 4 + day_type]
        // day_type: 0=weekday, 1=saturday, 2=sunday, 3=holiday
        data: Vec<f32>,
        edge_count: usize,

    **adaptive.rs:** Implement `AdaptiveWeights`:
    ```rust
    pub struct AdaptiveWeights {
        pub bpr_weight: f32,     // initial 0.40
        pub ets_weight: f32,     // initial 0.35
        pub hist_weight: f32,    // initial 0.25
        learning_rate: f32,      // 0.05 per update cycle
    }
    impl AdaptiveWeights {
        pub fn new() -> Self;
        pub fn update(&mut self, bpr_error: f32, ets_error: f32, hist_error: f32);
        pub fn blend(&self, bpr: &[f32], ets: &[f32], hist: &[f32]) -> Vec<f32>;
    }
    ```
    `update`: compute softmax-like adjustment -- decrease weight for high-error model, increase for low-error. Normalize to sum=1.0. Cap minimum weight at 0.05 to prevent any model from being completely zeroed.

    `blend`: weighted average per edge: `bpr_w * bpr[i] + ets_w * ets[i] + hist_w * hist[i]`.
