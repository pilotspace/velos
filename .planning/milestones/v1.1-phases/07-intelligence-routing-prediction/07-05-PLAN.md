  <action>
    Create `crates/velos-gpu/shaders/perception.wgsl` implementing the perception gather kernel:

    1. Define WGSL structs matching Rust-side (PerceptionParams, PerceptionResult, AgentState -- reuse struct layout from wave_front.wgsl).

    2. Bindings (SEPARATE bind group from wave_front):
    ```wgsl
    @group(0) @binding(0) var<uniform> params: PerceptionParams;
    @group(0) @binding(1) var<storage, read> agents: array<AgentState>;
    @group(0) @binding(2) var<storage, read> lane_agents: array<u32>;
    @group(0) @binding(3) var<storage, read> signals: array<SignalState>;
    @group(0) @binding(4) var<storage, read> signs: array<GpuSign>;
    @group(0) @binding(5) var<storage, read> congestion_grid: array<f32>;
    @group(0) @binding(6) var<storage, read> edge_travel_ratios: array<f32>;
    @group(0) @binding(7) var<storage, read_write> results: array<PerceptionResult>;
    ```

    3. `@compute @workgroup_size(256) fn perception_gather`:
    - Guard: `if (idx >= params.agent_count) { return; }`
    - **Leader detection:** Scan agents on same edge_id and lane_idx with higher position. Find closest one. Set leader_speed and leader_gap. If none found, leader_speed=0, leader_gap=9999.0.
    - **Signal state:** Read from signals array using edge_id as index (simplified -- assumes signal buffer indexed by edge). Set signal_state and signal_distance.
    - **Congestion (own route):** Read edge_travel_ratios[agent.edge_id]. This is current_travel_time / free_flow_travel_time.
    - **Congestion (area):** Convert agent position to grid cell: `cell_x = position_x / grid_cell_size`, `cell_y = position_y / grid_cell_size`. Read congestion_grid[cell_y * grid_width + cell_x].
    - **Signs:** Scan signs buffer for sign matching agent's edge_id. Set sign_speed_limit.
    - **Flags:** Set bit0 if congestion_own_route > 5.0 (severely congested = effectively blocked). Set bit1 if any emergency vehicle flag detected from agent flags scan.

    4. Write result to `results[idx]`.

    Note: Leader detection is simplified (linear scan of agents on same edge). For 280K agents this is acceptable because each agent only scans its own edge's agents (typically 1-20 vehicles per edge). The lane_agents sorted index from wave_front helps bound the scan.

    Validate with naga: `naga --validate crates/velos-gpu/shaders/perception.wgsl` (may need naga installed or use cargo test with wgpu validation).
