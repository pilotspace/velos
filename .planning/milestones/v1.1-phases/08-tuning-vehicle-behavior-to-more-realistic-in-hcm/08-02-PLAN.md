<objective>
Replace all hardcoded IDM and Krauss constants in wave_front.wgsl with per-vehicle-type parameters read from a uniform buffer, populated from VehicleConfig.

Purpose: Eliminate the GPU/CPU parameter mismatch (GPU IDM a=1.5/b=3.0 vs CPU a=1.0/b=2.0) by making both read from the same config. Every vehicle type gets its own parameters on the GPU.

Output: GpuVehicleParams struct, updated WGSL shader, binding 7 wired into dispatch.
</objective>
