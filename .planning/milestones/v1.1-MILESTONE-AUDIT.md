### Integration Architecture

Key architectural decisions enabling clean cross-phase integration:
- **Single SimWorld struct** owns all subsystems — wiring explicit through struct fields
- **Dirty-flag buffers** (signal_dirty, prediction_dirty) gate GPU uploads efficiently
- **Shared perception result buffer** between perception.wgsl (write) and wave_front.wgsl (read)
- **ArcSwap overlay** enables lock-free prediction reads during reroute evaluation
- **Profile encoding in GpuAgentState.flags** bits 4-7 enables GPU-side profile awareness
